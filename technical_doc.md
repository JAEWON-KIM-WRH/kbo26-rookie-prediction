## 3. 특성 엔지니어링

| 특성 | 설명 | 계산식 |
|------|------|--------|
| draft_score | 드래프트 순위 점수 | 1 / (round + pick * 0.1) |
| star_factor | 미디어 노출도 | media_exposure / 10 |
| injury_rate | 부상률 | injury_history * 0.1 |
| controversy_penalty | 논란 여부 | 0 or 1 |
| pitcher_score | 투수 성적 | (1/ERA) * (이닝/150) * (탈삼진/100) |
| batter_score | 타자 성적 | (타율/0.3) + (홈런/15) |

## 4. 모델 아키텍처

### 4.1 ML 모델
```python
RandomForest(n_estimators=100, max_depth=10, class_weight='balanced')
XGBoost(n_estimators=100, max_depth=6, scale_pos_weight=8)
GradientBoosting(n_estimators=100, max_depth=5)
SVM(kernel='rbf', class_weight='balanced')
```

### 4.2 DL 모델
```
MLP: Input → 64 → 32 → 16 → 1 (Sigmoid)
Attention: Input → Embedding → Self-Attention → FC → 1
```
